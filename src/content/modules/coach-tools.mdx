# Coach Tools

Essential tools for planning classes, managing training sessions, and tracking progress.

## Available Tools

### Round Timer
A simple, reliable timer designed specifically for grappling training sessions.

**Features:**
- Customizable round duration (minutes and seconds)
- Configurable rest periods
- Audio beeps for round transitions
- Automatic countdown before rounds
- Settings saved automatically

**Usage:**
1. Set your round duration (default: 5:00)
2. Set rest period (default: 0:20)
3. Choose number of rounds (default: 5)
4. Click Start to begin with a 3-2-1 countdown

**Tips:**
- Settings persist between sessions
- Use pause to take breaks during training
- Reset returns to initial settings
